ü´Å About This Research Project

LungHealth Check is an AI-powered lung health analysis platform developed as a research project at Daffodil International University. The project aims to improve early detection of lung abnormalities using multimodal deep learning techniques, combining X-Ray and CT scan analysis for more accurate diagnostics.
